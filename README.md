# Treino Web

Sistema web responsivo em HTML, CSS e JavaScript puro, com armazenamento local no navegador (`localStorage`).

## Áreas disponíveis

### Treinos
- Cadastro, edição e exclusão de treinos.
- Cor por treino e efeito cromático na página interna.
- Dia fixo, rotativo ou sem dia definido.
- Cadastro, edição e exclusão de exercícios.
- Grupo muscular por exercício.
- Cadastro, edição e exclusão de séries.
- Registro de carga, repetições e sensação do exercício.
- Histórico oculto por padrão.
- Histórico curto com últimos 3 dias.
- Expansão para histórico completo de até 12 registros.
- Após 12 registros, o mais antigo é substituído pelo novo.

### Dieta
- Cadastro, edição e exclusão de refeições.
- Macros por refeição: kcal, proteína, gordura e carboidrato.
- Cadastro de ingredientes com seus próprios macros.
- Marcação dos dias da semana em que cada refeição se aplica.
- Cadastro de refeições fora do padrão para dias específicos.
- Configuração de 1 ou 2 refeições livres semanais.
- Marcação de refeições concluídas.
- Cálculo de macros semanais.
- Contagem de dias sem furar a dieta.
- Selos progressivos de sequência: 3, 7, 15, 30 e 50+ dias.
- Botão “Furei dieta” para resetar a sequência da data selecionada.

### Controle de peso
- Cadastro, edição e exclusão de registros por data.
- Peso, altura e BF em cada registro.
- Cálculo automático de média semanal e mensal.
- Exibição do peso atual e médias na tela inicial.
- Cálculo aproximado de IMC com base no último registro.

## Instalação

Abra o arquivo `index.html` em qualquer navegador moderno.

## Ícones

O projeto já inclui:

- Favicon para navegador.
- Ícone Apple/mobile.
- Manifest para instalação como atalho/app no celular.

## Observações

Os dados ficam salvos somente no navegador usado. Se limpar o cache/dados do site, as informações podem ser apagadas.
